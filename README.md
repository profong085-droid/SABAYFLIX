# SabayFlix (សប្បាយហ្វ្លីច) 🍿

SabayFlix គឺជាកម្មវិធីមើលវីដេអូកម្សាន្ត (Movie Streaming Web Application) ដ៏ស្រស់ស្អាតនិងទាន់សម័យ ដែលបង្កើតឡើងដោយប្រើប្រាស់ **Next.js (App Router), Tailwind CSS, និង TypeScript**។ ការរចនារបស់វាត្រូវបានផ្តោតសំខាន់ទៅលើបទពិសោធន៍អ្នកប្រើប្រាស់ (Premium Cinematic UI) ស្រដៀងទៅនឹងកម្មវិធី Netflix ដែរ។

## 🚀 របៀបទាញយក និង ដំណើរការ (How to Clone and Run)

ដើម្បីអាចដំណើរការ Project នេះបាននៅលើកុំព្យូទ័ររបស់អ្នក សូមអនុវត្តតាមជំហានខាងក្រោមតាំងពីដើមដល់ចប់៖

### ជំហានទី១៖ ទាញយកកូដ (Clone the repository)
បើក Terminal (Command Prompt ឬ Git Bash) របស់អ្នក រួចវាយបញ្ជាខាងក្រោម ដើម្បីទាញយកកូដពី Github៖
```bash
git clone https://github.com/profong085-droid/SABAYFLIX.git
```
បន្ទាប់មក ចូលទៅកាន់ Folder របស់ Project នេះ៖
```bash
cd SABAYFLIX
```
*(បញ្ជាក់៖ ប្រសិនបើ Folder របស់អ្នកមានឈ្មោះផ្សេង សូមប្តូរឈ្មោះទៅតាមនោះ ឧទាហរណ៍៖ `cd "movie 1"`)*

### ជំហានទី២៖ តម្លើងកញ្ចប់បណ្ណាល័យ (Install Dependencies)
អ្នកត្រូវប្រាកដថាអ្នកបានតម្លើង Node.js រួចរាល់នៅក្នុងកុំព្យូទ័ររបស់អ្នក។ វាយបញ្ជាខាងក្រោមដើម្បីតម្លើង Packages ទាំងអស់ដែលគម្រោងនេះត្រូវការ៖
```bash
npm install
```

### ជំហានទី៣៖ ដំណើរការនៅលើកុំព្យូទ័រ (Run the Development Server)
បន្ទាប់ពីតម្លើងរួចរាល់ សូមវាយបញ្ជាខាងក្រោមដើម្បីរត់កម្មវិធី៖
```bash
npm run dev
```
រួចបើកកម្មវិធី Browser របស់អ្នក (Chrome, Safari...) ហើយចូលទៅកាន់លីង៖ **[http://localhost:3000](http://localhost:3000)** ដើម្បីមើលលទ្ធផលពិត!

## 🌐 របៀបបង្ហោះទៅកាន់ Vercel (How to Deploy on Vercel)

Vercel គឺជាកន្លែងដ៏ល្អ និងងាយស្រួលបំផុតក្នុងការបង្ហោះ Next.js Project ដោយឥតគិតថ្លៃ។ ខាងក្រោមនេះជារបៀបបង្ហោះវ៉ិបសាយរបស់អ្នកឲ្យគេអាចចូលមើលបានទូទាំងពិភពលោក៖

### វិធីទី១៖ បង្ហោះតាមរយៈ Vercel Dashboard (ងាយស្រួលបំផុត និងណែនាំឲ្យប្រើ)
1. ត្រូវប្រាកដថាអ្នកបានបញ្ជូន (Push) កូដរបស់អ្នកទាំងអស់ទៅកាន់គណនី **GitHub** របស់អ្នករួចរាល់។
2. ចូលទៅកាន់គេហទំព័រ [vercel.com](https://vercel.com/) រួចចុះឈ្មោះចូល (Sign up / Login) ដោយប្រើគណនី GitHub របស់អ្នក។
3. នៅលើផ្ទាំង Dashboard របស់ Vercel សូមចុចលើប៊ូតុង **"Add New..."** រួចជ្រើសរើសយកពាក្យ **"Project"**។
4. ស្វែងរកឈ្មោះ Project SabayFlix របស់អ្នកនៅក្នុងបញ្ជីកូដ GitHub នោះ រួចចុចប៊ូតុង **"Import"** នៅក្បែរវា។
5. មិនចាំបាច់កែប្រែការកំណត់ (Settings) អ្វីទាំងអស់ សូមអូសចុះក្រោមរួចចុចប៊ូតុង **"Deploy"** តែម្តង។
6. រង់ចាំប្រហែល ១ ទៅ ២នាទី ម៉ាស៊ីននឹងតម្លើងឲ្យអ្នកដោយស្វ័យប្រវត្តិ។ ពេលចប់ អ្នកនឹងទទួលបានលីងវ៉ិបសាយពិតប្រាកដ (Live URL) ដែលអ្នកអាចថតចម្លង (Copy) ផ្ញើឲ្យអ្នកដទៃចូលមើលបានភ្លាមៗ! 🎉

### វិធីទី២៖ បង្ហោះតាមរយៈ Vercel CLI (សម្រាប់អ្នកសរសេរកូដដែលចូលចិត្ត Terminal)
បើក Terminal របស់អ្នក រួចតម្លើង Vercel ជាសកល (Global) ៖
```bash
npm i -g vercel
```
រួចវាយបញ្ជាដើម្បីបង្ហោះកូដ៖
```bash
vercel
```
បន្ទាប់មក វានឹងលោតសួរអ្នកនូវសំនួរមួយចំនួន គ្រាន់តែចុច Enter (យល់ព្រម ឬ Y) តាមការណែនាំរហូតដល់ចប់ជាការស្រេច។
